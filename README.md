# Sistema-de-calculo-de-tarifa-de-energia
Avaliação do curso técnico em Inteligência Artificial
# Apresentação Oral sobre Python

Este repositório contém a estrutura da apresentação, os tópicos dos slides e o roteiro de fala para uma apresentação oral sobre Python.

## Objetivo

Apresentar um sistema, em Python, que irá calcular o valor a ser pago em uma conta de energia em Manaus de forma clara, organizada e didática, destacando seus conceitos básicos, aplicações e vantagens.

## Organização da apresentação

### 1. Abertura
- Saudação inicial.
- Apresentação do tema.
- Explicação rápida do objetivo da fala.

### 2. Conteúdo principal
- Como calcular uma conta de energia.
- Principais características.
- Onde é usado.
- Exemplo prático em Python.

### 3. Encerramento
- Recapitulação dos pontos principais.
- Mensagem final.
- Espaço para perguntas.

## Estrutura dos slides

1. Título e identificação.
2. O Ponto de Partida: Tarifa B1
3. O Peso dos Tributos
4. A Lógica da COSIP
5. O Algoritmo de Cálculo da bandeira tarifária
6. Exemplo de código.
7. Conclusão.

## Roteiro resumido de fala

Python é uma linguagem conhecida por sua sintaxe simples, o que facilita o aprendizado e o desenvolvimento de projetos. Ela é usada em automação, análise de dados, desenvolvimento web e inteligência artificial. Por isso, é uma das linguagens mais populares da atualidade.

## Observações

- Mantenha os slides objetivos.
- Use linguagem simples.
- Treine a apresentação antes de apresentar.
- Se possível, inclua uma pequena demonstração de código.
